2013-07-2

设Ｍ，Ｘ，Ｒ，Ｆ，Ｃ五个功能键
Ｍ：功能开关，按下键按　电阻（Ｒ）－〉电容（Ｃ）－〉电感（Ｌ）顺次轮换切换，长按功能键则进入系统调置（Ｓ）模式。
Ｘ：串并联切换开关，按下键按　串联（Ｚ）－〉并联（Ｐ）顺次轮换切，长按（１秒以上）则进入记忆功能，记录当前测量值，并以%分比显示与下一测量的差异，在记忆功能下，短按则回到串联。
Ｒ：量程开关，按下键按　４０－〉１Ｋ－〉１０Ｋ－〉１００Ｋ 　顺次轮换切换，长按（１秒以上）则直接进入自动，在自动状态下，短按则回到４０。
Ｆ：频率开关，按下键按　１００－〉１Ｋ－〉７.８Ｋ　顺次轮换切换。
Ｃ：清零，当表笔开路时，开路清零，当表笔短路清零，　注意短按只清当前量程的，长按（１秒以上）则清所有量程。
